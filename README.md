# CorDecor Creative Assistant

## Project Overview

CorDecor Creative Assistant is an AI-powered web application developed as part of the AI Skills Accelerator project. It is designed to help décor and event styling businesses streamline their workflows through AI-assisted client communication, event setup planning, and creative décor inspiration. The application brings these capabilities together in a single, easy-to-use dashboard.

## Features Implemented

- **AI Client Email Generator** — Generate professional, editable client emails for enquiries, bookings, quotations and follow-ups.
- **AI Setup Planner** — Produce structured event setup plans covering venue layout, décor schedule, checklist and supplier notes.
- **AI Creative Inspiration Studio** — Discover themed décor ideas, colour palettes and styling suggestions tailored to event requirements.
- **Responsive Dashboard and Navigation** — A clean, responsive layout with sidebar navigation that works across desktop and mobile devices.
- **Responsible AI Guidance** — Built-in guidance reminding users to review and verify AI-generated content before sharing it with clients.
- **Copy and Regenerate Functionality** — Quickly copy generated outputs to the clipboard or regenerate fresh suggestions with one click.
- **Clean Branded User Interface** — A polished, professional interface styled to reflect the CorDecor brand identity.

## Technologies and Tools Used

- **Lovable AI** — Used to build and refine the application UI and functionality.
- **React** — JavaScript library for building the user interface.
- **TypeScript** — Typed superset of JavaScript used for safer, more maintainable code.
- **Vite** — Fast build tool and development server.
- **Tailwind CSS** — Utility-first CSS framework for styling.
- **GitHub** — Version control and project repository hosting.
- **ChatGPT** — Used for prompt engineering, planning and documentation.

## Setup Instructions

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/RoelienCornelius/cordecor-creative-assistant.git
   cd cordecor-creative-assistant
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open the application in a browser**
   Navigate to the local URL shown in the terminal (typically `http://localhost:8080`).
